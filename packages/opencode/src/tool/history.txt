Search raw conversation history when curated `memory` is not enough.

Prefer `memory` first. Use `history` for verbatim recall, old tool traces, or cases where memory has no useful hit.

Actions:
- `search`: full-text search over raw messages and tool traffic.
- `around`: fetch nearby messages for a specific `message_id`.

Default scope is the current project; use `scope="global"` to search across projects.
